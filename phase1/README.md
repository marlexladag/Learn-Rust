# Phase 1: Rust Foundations

Welcome to Phase 1! By the end of this phase, you'll understand Rust's basic syntax and be able to write simple programs.

## Contents

## Exercises

After completing the tutorials, test your knowledge with the [exercises](exercises/).

## How to Use This Tutorial

1. Read each section's `README.md` for explanations
2. Run the example code with `cargo run` in each directory
3. Modify the examples to experiment
4. Complete the exercises at the end

## Running Examples

Each section is a Cargo project. To run:

```bash
cd 01_setup
cargo run
```
